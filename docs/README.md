## Documentation Maintenance

This documentation is maintained alongside the codebase. When making changes:

1. **Update relevant documentation** when code changes
2. **Test all code examples** in documentation
3. **Verify all links** after moving or renaming files
4. **Keep examples synchronized** with the codebase

For detailed guidelines on maintaining documentation, see the [Documentation Maintenance](developer/contributing.md#documentation) section in the Contributing guide, or refer to [AGENTS.md](../AGENTS.md) for AI agent-specific guidelines.
